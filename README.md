# Sentiment Analysis of News Headlines Using NLP

## Project Overview
This project analyzes news headlines and predicts whether the sentiment is positive, negative, or neutral using Natural Language Processing (NLP) and Machine Learning techniques.

## Features
- News headline sentiment prediction
- Data preprocessing using NLP
- Machine learning model training
- JSON dataset handling
- Simple Python implementation

## Technologies Used
- Python
- Pandas
- NLTK
- Scikit-learn

## Dataset
Sarcasm Headlines Dataset (JSON format)

## Files Included
- app.py
- main.py
- test.py
- Sarcasm_Headlines_Dataset.json
